GRAFCETJSON Viewer

GRAFCETJSON Viewer es una herramienta para la creacin, edicin y simulacin de diagramas GRAFCET construida en la librera de diagramado GoJS, desarrollada por Northwoods Software, los cuales amablemente proporcionaron una licencia acadmica para el desarrollo de este software.

Existen mltiples herramientas para el diseo GRAFCET y la propuesta de Northwoods Software incluye un flujo de trabajo diferente e interesante que nos impuls a seguirla

En la librera de GoJS existen dos elementos principales: Nodos y enlaces. En GRAFCETJSON, dentro de los nodos podemos encontrar etapas y elementos de divergencia/convergencia. Y en los enlaces podemos encontrar los enlaces (enlaces directo de arriba hacia abajo, saltos de secuencia hacia atrs, saltos de etapas) y las transiciones.

GRAFCETJSON Viewer deja a un lado las interfaces de "pick-and-place" o drag-and-drop para utilizar un sistema build-down, donde el usuario disea el sistema de forma descendiente seleccionando que elemento procede a otro a medida que la herramienta coloca automtica e inteligentemente los elementos que los conectan.

-          Para borrar una accin, se debe dar doble clic en la accin y darle clic al botn Delete

Expresiones booleanas y receptividades

Las expresiones booleanas se escriben directamente en texto plano. Estas expresiones se pueden escribir tanto en transiciones como para condicionar acciones. Es posible usar expresiones simplificadas mediante parntesis, sin embargo, por simplicidad, solo es posible negar variables, no expresiones enteras (para circunvenir esta limitante se recomienda convertir la expresin a una equivalente donde solo se nieguen variables).

Para expresar variables negadas se escribe el prefijo - con el teclado antes de la variable a negar (Ej. -S1).

Para expresar variables con flanco de subida o bajada se colocan los prefijos RE- o FE- respectivamente (Ej. RE-S1*FE-S2).

Las variables se crean automticamente al agregar acciones y transiciones.

Preguntas

Cmo copio una etapa?

Seleccionar la etapa > Clic + Ctrl +Arrastrar, al editar la copia o la original, ambas etapas tendrn los mismos elementos automticamente hasta que se guarde el GRAFCETJSON.

Cmo borro un enlace?

Seleccionar el enlace > Suprimir.

Cmo borro una transicin?

Doble clic en el enlace > Dentro del men de configuracin, Clic en el botn Delete.

Cmo cambio el tipo de variable?

Una vez creada, no se puede cambiar el tipo de una variable. Es necesario borrar la variable del sistema y crearla de nuevo.

Cmo borro una variable del sistema?

Desde la ventana de variables del sistema, dar Clic en la equis superior derecha mientras no se est simulando.

Cmo cierro una secuencia divergente o secuencia selectiva/simultnea?

Desde de una de las ramas, aada el nodo al que han de converger. Para conectar las dems ramas, seleccione el ltimo nodo de cada rama y conctelos con un enlace directo a el nodo de convergencia.
